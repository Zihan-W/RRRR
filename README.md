# RRRR Paper Website

This is the project website for the paper **"RRRR: Rapid Real-World Residual RL for Multi-Task VLA Adaptation"**.

## Features
- Built with **React** and **TypeScript**
- Styled with modern, responsive **CSS**
- Scaffolding by **Vite**

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```

## Deployment
You can deploy the `dist` folder to GitHub Pages or any other static hosting service.
